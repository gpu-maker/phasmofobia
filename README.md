# Haunt Investigator

Browser-based ghost investigation horror game using HTML, CSS, and Vanilla JavaScript.

## Features
- Pseudo 3D raycast engine
- Volumetric fog
- Ghost AI
- EMF reader
- Thermometer device
- Room temperature simulation
- Procedural house generation
- Procedural audio + textures (no assets required)

## Controls
W — Forward
S — Backward
A/D — Turn

## Run
Open index.html in browser.
